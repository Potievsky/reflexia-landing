# Reflexia landing draft

## Що це

Перший робочий draft лендингу для продажу Reflexia як туристичного інвестоб'єкта у Славському.

Файл сторінки:

- `index.html`

Локально відкривати можна через простий HTTP server з цієї папки.

## Що потрібно підставити перед публікацією

- Meta Pixel ID.
- GA4 ID, якщо потрібна окрема аналітика.
- Контакти: телефон, Telegram, email.
- Фінальне формулювання ціни і сценаріїв продажу.
- Юридично підтверджений опис документів і структури угоди.

## Поточна логіка сторінки

- Основне позиціонування: не готовий готель, а інвестоб'єкт на стадії високої готовності.
- Основний режим повідомлення: `Інвестору`.
- CTA: отримати інвест-пакет / обговорити купівлю.
- Форма відправляє заявки в Google Forms / Google Sheets і фільтрує лід за сценарієм і бюджетом.

## Перевірено

- Desktop viewport: 1280 x 720.
- Mobile viewport: 390 x 844.
- Битих зображень не виявлено.
- Відео завантажується.
- Горизонтального скролу на mobile не виявлено.
